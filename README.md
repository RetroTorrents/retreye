# retreye
From the kilobytes to the gigabytes, the Retreye (Retro Eye) unites all ROM sites to save video games!

Disclaimer: You will not find torrent magnet links here, please visit the official RetroTorrents site!

## Frequently Asked Questions (FAQ)

### What is the Retreye?

The Retreye is a rom torrent index, this public repository is to:
- Suggest and set standards of the folder structure inside every torrent in specific platforms.
- Add and update ROMs to the database, at the request of volunteers and seeders.

### How does it work and why is this needed?

The Retreye functions as a mirror for others to deploy on their own domain name, which increases resilience and mitigates downtime in the event the main site becomes unavailable.

For example, the name can be seen in the following examples:

- retreye.minerva-archive.org (Minerva Archive)
- retreye.retrotorrents.net (RetroTorrents)
- retreye.erista.me (Erista Group)
- retreye.vimm.net (Vimm's Lair)

Due to github's TOS, the Retreye can only be downloaded through our website as a torrent.
